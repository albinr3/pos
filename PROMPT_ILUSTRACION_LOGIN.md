# Prompt para Ilustración Decorativa - Página de Login MOVOPos

## Descripción General
Crea una ilustración decorativa moderna y profesional para una página de login de un sistema POS (Point of Sale) e inventario. La ilustración debe ser sutil, elegante y complementar el diseño sin distraer de los elementos de autenticación.

## Estilo Visual
- **Estilo**: Moderno, minimalista, tech/negocio
- **Paleta de colores**:
  - Primarios: Gradiente de púrpura a azul (#060097 → #8204FF → #C10FFF)
  - Secundarios: Blancos translúcidos, grises suaves
  - Acentos: Tonos de púrpura claro y azul eléctrico
- **Técnica**: Ilustración vectorial o digital plana (flat design) con toques sutiles de profundidad
- **Ambiente**: Profesional, tech-friendly, empresarial

## Elementos Temáticos
La ilustración debe incluir (de forma sutil y abstracta):
1. **Iconos de tecnología/comercio**:
   - Pantallas de punto de venta estilizadas
   - Gráficos de inventario (barras, tablas sutiles)
   - Iconos de productos/paquetes minimalistas
   - Gráficos de crecimiento (flechas ascendentes, gráficos de línea)

2. **Elementos abstractos**:
   - Formas geométricas flotantes (círculos, cuadrados suaves)
   - Líneas de conexión sutiles
   - Partículas/estrellas pequeñas dispersas
   - Ondas suaves de fondo

3. **Concepto**:
   - Transmisión de datos/digitalización
   - Control y organización (representado por elementos ordenados)
   - Eficiencia y productividad

## Composición
- **Posición**: Lado derecho o izquierdo del formulario de login
- **Orientación**: Vertical o cuadrada, adaptable
- **Espacio negativo**: Abundante, no debe saturar
- **Enfoque**: Elementos centrales más definidos, bordes más sutiles

## Detalles Técnicos
- **Formato**: SVG preferible, PNG con fondo transparente aceptable
- **Resolución**: Alta resolución para pantallas Retina (mínimo 800x800px, preferible 1200x1200px)
- **Tamaño del archivo**: Optimizado (<500KB preferiblemente)
- **Transparencia**: Fondos transparentes donde aplique
- **Escalabilidad**: Debe verse bien en diferentes tamaños (responsive)

## Características Específicas
- **Colores principales**: Usar el gradiente púrpura-azul como color dominante
- **Transparencias**: Usar capas con diferentes opacidades (10-30%) para profundidad
- **Blur sutil**: Algunos elementos de fondo pueden tener ligero desenfoque
- **Gradientes**: Transiciones suaves entre los colores de la paleta
- **Líneas**: Delgadas (1-2px), suaves, sin bordes duros

## Restricciones
- **NO** incluir texto o números legibles
- **NO** usar colores que no estén en la paleta
- **NO** crear elementos que puedan confundirse con botones o elementos interactivos
- **NO** saturar el espacio con demasiados elementos

## Mood/Atmósfera
- Moderno y profesional
- Tecnológico pero acogedor
- Limpio y organizado
- Innovador y eficiente

## Variaciones Sugeridas
1. **Versión con personaje**: Un personaje minimalista (silueta simple) trabajando con tecnología
2. **Versión abstracta pura**: Solo formas geométricas y elementos abstractos
3. **Versión híbrida**: Combinación de elementos abstractos con iconos sutiles de comercio/tecnología

## Ejemplos de Inspiración
- Ilustraciones de Stripe (clean, tech-forward)
- Diseños de Figma Community (moderno, vectorial)
- Ilustraciones de apps de fintech (profesional, elegante)

## Prompt Final Recomendado para IA Generativa

```

Style: Clean flat design with subtle depth, professional tech aesthetic, vector-based

Color palette: Gradient from deep purple (#060097) through vibrant purple (#8204FF) to electric purple-blue (#C10FFF), with translucent whites and soft grays as accents

Elements: Abstract geometric shapes (circles, soft squares), subtle tech icons (stylized POS screens, inventory charts), soft floating particles, gentle connecting lines, waves or data flow representations

Composition: Vertical or square format, abundant negative space, central focus with softer edges, transparent background where applicable

Mood: Modern, professional, tech-friendly, organized, efficient, innovative yet welcoming

Technical: High resolution (1200x1200px), SVG preferred or PNG with transparency, optimized file size, scalable design

Restrictions: No readable text or numbers, use only specified color palette, avoid interactive-looking elements, keep composition uncluttered

Atmosphere: Clean, tech-forward, business-oriented, sophisticated
```

## Notas Adicionales
- La ilustración debe complementar el diseño existente sin competir por atención
- Debe funcionar tanto en modo claro como oscuro (considerar variaciones)
- Mantener coherencia con el estilo del sitio web (hero-img.svg como referencia de estilo general)
